# CORS Activity
Atividade para testar o funcionamento do CORS. 

## Instruções de Execução

### Backend

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor Node.js:
```bash
node server.js
```

O backend estará rodando na porta padrão 8080.

### Frontend

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
serve -l 3000
```

O frontend estará disponível em `http://localhost:3000`.

## Notas

- Certifique-se de que ambos os servidores estão em execução para testar a integração CORS completa.
- Verifique se as portas 3000 e a 8080 não estão em uso por outros processos.
